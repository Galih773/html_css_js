BEM: Block Element Modifier

BEM is a naming convention for CSS classes that helps to keep CSS organized and maintainable.

Block: A standalone component that can be used independently of the rest of the page.

Element: A part of a block that can be used independently of the rest of the block.

Modifier: A variation of a block or element that can be used independently of the rest of the block or element.

Example:

Block: .header

Element: .header\_\_logo

Modifier: .header\_\_logo--red
